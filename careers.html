MARIE.js

Why not join us?

Our team consists of volunteer developers who develop MARIE.js. We are currently looking for:

  • Developers
    • User Experience Developers
    • ES6 migration Developers
  • Documentation Writers
It will look good on your resume/CV too!

Developer

Do you love programming? Then this role will suit you.

You must:
  • Have previous programming experience
  • Be willing to learn new languages
  • If you learnt HTML5, CSS3 and JavaScript (ES6) will be beneficial
  • Knowledge of Google Drive APIs will also be beneficial
This role requires you to:
  • Maintain and fix issues arising from Public Submissions
  • Test out code (especially complicated ones) and solve issues arising from them
  • Migrate code to ES6 for future developers
To join us simply pull from our GitHub repository and make a pull request.

Documentation Writer

Do you love programmming but don't have the capacity to do so? Then this role will suit you.

You must:
  • Have programming experience
  • Used MARIE.js before and have a in-depth understanding of MARIE Assembly language
This role requires you to:
  • Develop Documentation and New Material such as New Tutorials
  • Maintain and report issues arising from example codes
  • Test out code (especially complicated ones)
To join simply talk or email us with the HEADER/topic: Doc Writer
